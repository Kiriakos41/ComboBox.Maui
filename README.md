Maui.ColorPicker ⬇️

a custom control for .net maui ComboBox

Getting Started
**NOTE**
add namespace
```xml
xmlns:controls="clr-namespace:ComboBox.Maui;assembly=ComboBox.Maui"
```
**NOTE**
create control
```xml
<controls:ComboBox ItemsSource="{Binding Items}" ShownText="Name" />
```
